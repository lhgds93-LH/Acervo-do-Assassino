O Acervo do Assassino — Sala 1 v24.36 PWA

Como testar:
1. Extraia o ZIP.
2. Publique a pasta em GitHub Pages, Firebase Hosting ou outro servidor HTTPS.
3. Abra o index.html pelo navegador.
4. No celular, use "Adicionar à tela inicial".

Observação:
- O PWA inclui index.html, manifest, service worker e ícones.
- Para instalação como app, use HTTPS ou localhost.
